Use Scenarios

As a result of the Elixir BioHackathon 2020 we defined the concept maps for 5 possible use scenarios:

Map 1: Primary analysis of genome sequence: genome assembly, gene prediction and annotation
This map describes the basic steps for genome assembly starting from NGS data, gene prediction and functional/structural annotation of encoded proteins.

Map 2: Multiple Sequence Alignment and Phylogeny
This map organizes tools and resources to perform comparison among genes or genomes and their phylogenetic analysis.

Map 3: Mutation variant types
This map organizes tools/resources for identification of emerging variations, for variation annotation in terms of type, rate and fitness and for mapping variations on proteins (sequence or structure).

Map 4: Assessing antibiotic resistance from genome sequence
This map organizes tools and resources that allow to highlight specific genes or variations that can be related to important phenotypes such as antimicrobial resistance.

Map 5: Important factors for vaccine design
This map describes tools and resources for protein structure modelling, prediction of exposed proteins, detection epitopes useful for vaccine design.
